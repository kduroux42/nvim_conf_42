# nvim_conf_42
Add nvim for 42 Mac/DEL\
get nvim appimage, extract and move in $HOME\
move nvim in .config\
and do alias
### Install :
```
bash -c "$(curl -fsSL https://raw.github.com/kduroux42/nvim_conf_42/master/install.sh)"
```
### What you need to do :
change your name in header plugin
##### How to do : 
- `nvim` in a terminal
- press `c`
- go in `lua/plugins/header`
- change your name ^^
